The official blog for MorrisionAgenda font is here!

MorrisonsAgenda, Morrisons' branding font, is now available on GitHub for everyone to try them out and download.

Morrisons has been using MorrisonsAgenda font since 2016 - in branding, its advertisements and more. (see left of image below for the usage of this font)

Mtaylor848, CC BY-SA 4.0, via Wikimedia Commons

We offer three weights (light, medium and semibold) as webfont and desktop fonts, but there will be plans to add bold weight.

In the Morrisons' website, the font weight allocated are on the table below. The allocated weights is also used in this website.

Font name Allocated weight
Light 400 (normal)
Medium 600 (semibold)
Semibold 700 (bold)

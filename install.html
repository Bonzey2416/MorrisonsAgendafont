MorrisonsAgenda font

Installation

You can add the stylesheet for this font in WOFF format to websites.

<link rel="stylesheet" href="https://bonzey2416.github.io/MorrisonsAgendafont/fonts.css">

You can now install fonts in TTF (for modern Windows, macOS and Linux desktops) and TTF.BIN (for older System 7 to Mac OS 9.2.2 desktops) file formats.

Install TTF files (v1.1) (57.2 kB) Install TTF.BIN files (v1.1) (65 kB)
For Windows 10 and 11

Extract the installed fonts. Go to Settings > Personalisation > Fonts. Drag and drop the fonts to the drag and drop space and this will install the fonts.

For Windows 8.1 and older

Extract the installed fonts. Right-click the fonts and select Install. This will install the fonts.

For macOS (10.x and later)

Extract the installed fonts. Drag and drop the fonts to the Font Book app icon to install the fonts.

For System 7 to Mac OS 9.x

Extract the installed fonts using a extraction software like Stuffit Expander to get the font suitcases. Open the hard disk folder, then open System folder, then open Fonts folder. Drag and drop the fonts to the Fonts folder and this will install the fonts. Close any running applications before installing.

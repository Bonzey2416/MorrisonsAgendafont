MorrisonsAgenda font

Version 1.1 of our MorrisonsAgenda font is now available! Download it now!

Overview

This font family contains three weights and supports kerning.

Install View specimen

Available in WOFF (online webfont), TTF (for modern Windows, macOS and Linux desktops) and TTF.BIN (for older System 7 to Mac OS 9.2.2 desktops) file formats.

History and development

This font was originally designed by Morrisons in 2016 to replace the branding font. 8 years later - this font is finally here on GitHub and improved. It is originally based on the original Agenda font, which was created by Greg Thompson in 2001.
